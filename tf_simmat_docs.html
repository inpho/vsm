simmat_docs¶

TfViewer.simmat_docs(docs)¶

Calculates the similarity matrix for a given list of documents.

Parameters:docs (list, optional) – A list of documents whose similarity matrix is to be computed. Default is all the documents in the model.
Returns:an IndexedSymmArray object n x n matrix containing floats where n is the number of documents. considered.
See Also:vsm.viewer.similarity.simmat_docs()
