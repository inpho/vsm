BaseCorpus.view_metadata(ctx_type)¶

Displays the metadata corresponding to a tokenization of the corpus. This method can be used in Corpus as well as BaseCorpus

Parameters:ctx_type (string-like) – The type of a tokenization.
Returns:The metadata for a tokenization.
